A.	How to run each .asm file
1.	Open MARS
2.	Load .asm files (version 1, 2, 3)
3.	Click Assemble on the top or click F3
4.	Click run or click F5
5.	Enter the password when prompted

B.	How to check correct password
•	After assembling and running a program, it will give you a prompt, “Enter Password:”. If you enter right password in, it will output “Access Granted”. Anything else with give “Access Denied”.
1.	Version 1
-	Enter abcd1234 = Access Granted
-	Enter anything else = Access Denied
2.	Version 2
-	Enter qwer1234 = Access Granted
-	Enter anything else = Access Denied
3.	Version 3
-	Enter comp3350 = Access Granted
-	Enter anything else = Access Denied

C.	How to dump .text and .data
1.	Open MARS
2.	Load .asm file (version 1, 2, 3)
3.	Click Assemble
4.	Click file and dump memory 
5.	Segment: .text   Format: Binary or Segment: .data   Format: Binary
6.	Save as version#_data.bin or version#_text.bin
